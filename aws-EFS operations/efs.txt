# Amazon EFS with Multiple EC2 Instances

## Overview
This project demonstrates Amazon Elastic File System (EFS) mounted on two EC2 instances, allowing shared file access.

## Services Used
- Amazon EC2
- Amazon EFS
- Amazon VPC
- Security Groups

## Steps Performed
1. Created an Amazon EFS file system.
2. Launched two EC2 instances in the same VPC.
3. Allowed NFS traffic on port 2049.
4. Mounted the same EFS on both instances.
5. Added content from both servers.

## Verification
- Server1: `this is server1`
- Server2: `this is server2`
- Both lines visible on both instances.

## Conclusion
Amazon EFS provides shared, persistent storage across multiple EC2 instances.
