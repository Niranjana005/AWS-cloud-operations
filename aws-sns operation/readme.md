**ABOUT SNS**



AWS Simple Notification Service (SNS) is a fully managed messaging service that allows you to send notifications to subscribers like:



&nbsp;                     Email

&nbsp;                     SMS

&nbsp;                     HTTP/HTTPS endpoints

&nbsp;                     Lambda functions

&nbsp;                     SQS queues

**DESCRIPTION**



This repository contains AWS SNS message creation operation that I have created for testing and learning purpose. And this message are stored in the text format for further references.



**TOOLS REQUIRED**



&nbsp;       AWS-SNS



**OPERATIONS DONE**

&nbsp;

&nbsp;     1).In the SNS section of the AWS, in the topic section, a topic is created with a name and type used

&nbsp;     2).And then subscription is created for the topic created with the type of message here EMAIL is given so along with that the EMAIL ID is given so we can receive the mail

&nbsp;     3).And after the subscription, the no-reply mail from the AWS is sent to make us for the subscription 

&nbsp;     4).And after the subscription mail the topic will be published and there the message that need to be sent in the mail will be created with a name tag will be created

&nbsp;     5).And then the mail will be received as per the name tag and the content that we typed
